Projects

PADES Printing

I developed a business website for PADES Printing using HTML, CSS, JavaScript, React, and Bootstrap. The website features images provided by the company and icons from free platforms like Icons8. Wireframing was done in Figma, and the website is hosted on GitHub Pages with a domain from Hostinger.

Nutri Forage

This project highlights my skills in building Android apps using Java and XML, with Firebase for backend support. By using Figma for design, I created a user-friendly interface and integrated it seamlessly with Firebase.

Bookstore App
